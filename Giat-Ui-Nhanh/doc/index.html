Tổng Quan Hệ Thống

Giặt Ủi Nhanh là một website dịch vụ giặt ủi chuyên nghiệp, cung cấp các dich vụ như: giặt sấy nhanh; giặt ủi lấy liền; giặt hấp giao tân nơi; giặt hấp áo vest, váy cưới; giặt khăn khách sạn, nhà hàng, Spa; vệ sinh giày, giặt giày cao cấp và nhận đặt lịch tư vấn trực tuyến 24/7.

Lưu ý: Hệ thống được xây dựng với PHP,HTML,CSS, JS, MySQL, Bootstrap 5, XAMPP cung cấp giao diện thân thiện và chức năng quản lý thông tin đặt lịch.

Yêu Cầu Hệ Thống

  • Phần mềm máy chủ: XAMPP (khuyến nghị)
  • PHP 7.4 trở lên (đã tích hợp sẵn trong XAMPP)
  • MySQL 5.7 trở lên / MySQL 8.0 (XAMPP)
  • Apache Web Server (có sẵn trong XAMPP)

Cấu Trúc Database

1. Bảng User

Bảng users
id int(11) ID tự tăng, khóa chính
user_name varchar(150) Tên người dùng
user_tel int(11) Số điện thoại người dùng
user_email varchar(250) Email đăng nhập
role enum('admin') Vai trò
user_password varchar(250) Mật khẩu
user_password varchar(250) Mật khẩu
created_at timestamp Thời gian tạo
updated_at timestamp Thời gian cập nhật

2. Bảng Service

Bảng services
id int(11) ID tự tăng, khóa chính
service_name varchar(150) Tên dịch vụ
summary text Mô tả ngắn về dịch vụ
service_price double Giá dịch vụ
created_at timestamp Thời gian tạo
updated_at timestamp Thời gian cập nhật

3. Bảng Order

Bảng orders
id int(11) ID tự tăng, khóa chính
customer_name varchar(50) Tên khách hàng
customer_tel int(11) Số điện thoại khách hàng
customer_address varchar(150) Địa chỉ khách hàng
service_id int(11) ID dịch vụ (khóa ngoại)
total_price double Tổng tiền đơn hàng
note text Ghi chú của khách hàng
order_status enum('Pending','Processing','Completed','Cancel') Trạng thái đơn hàng
transaction_status enum('UnPaid','Paid') Trạng thái thanh toán
created_at timestamp Thời gian tạo
updated_at timestamp Thời gian cập nhật

Hướng Dẫn Cài Đặt

Trạng Thái Đơn Hàng
1

Chuẩn Bị Môi Trường (XAMPP)

Tải và cài đặt XAMPP để chạy website PHP trên máy tính.

2

Đưa Source Code Vào XAMPP

  1. Copy source code đã tải xuống
  2. Dán vào thư mục: C:\xampp\htdocs\
  3. Giải nén file source code
3

Tạo Database Trên phpMyAdmin

  1. Mở trình duyệt và truy cập vào url: http://localhost/phpmyadmin
  2. Sau đó tạo database.
  3. Nhấp vào database đã tạo
  4. Nhấp vào nút Nhập ở thanh menu
  5. Nhấp vào Chọn tệp
  6. Chọn vào file SQL trong source code đã giải nén.
  7. Nhấp vào nút Nhập.
4

Cấu Hình Kết Nối Database

Mở file app/core/Database.php và chỉnh như sau(nếu đúng thì giữ nguyên):

private $host = "localhost";
private $db = "giat-ui-nhanh";
private $user = "root";
private $pass = "";

5

Chạy Website


Truy cập vào Website
  1. Mở trình duyệt và truy cập vào url: http://localhost/Giat-Ui-Nhanh/
  2. Website hiển thị là cài đặt thành công
Admin Dashboard
  1. Truy cập vào url: http://localhost/Giat-Ui-Nhanh/?ctrl=page&act=dashboard
  2. Email: admin@gmail.com
