# Website Dịch Vụ Vệ Sinh – PHP & MySQL

## Giới thiệu
Dự án được thực hiện trong quá trình thực tập, xây dựng website đặt lịch dịch vụ vệ sinh với PHP thuần và MySQL.

## Chức năng chính
- Khách hàng đặt lịch dịch vụ vệ sinh
- Quản lý khách hàng
- Quản trị viên duyệt / hủy đơn hàng
- Hỗ trợ xuất hóa đơn điện tử

## Công nghệ sử dụng
- PHP
- MySQL
- HTML / CSS / JavaScript

## Tài liệu chi tiết
Tài liệu hệ thống (database, logic PHP, hướng dẫn sử dụng) nằm tại: docs/index.html hoặc https://huynhdat1810.github.io/vesinhcare/docs/index.html
